# 职友网爬虫说明

---

**第三方模块**：Scrapy<br>

**项目说明**：
&emsp;&emsp;职友网是义乌荟英人力资源有限公司创办的公司网站，一家面向大型公司和快速发展的中小企业提供一站式专业人力资源服务网站。
&emsp;&emsp;这个爬虫主要是爬取职友网站所有的公司信息，包括(公司名称、地址、简介、平均薪资、电话...)<br>

**为什么要采集该网站数据**
- 该站点含有企业点评信息,薪资结构等重要数据		
- 该网站数据量足够大，可以用于后续使用

**代码说明**:

- 目标：获取职友集网站上的企业信息
- 功能：
   - 企业基础信息	
   - 融资信息		
   - 排名信息		
   - ……..
- 已知反爬手段及应对策略
    - 暂无反爬措施
    - 不过这个网站，不可以处理高频率请求

<br>
注意点：本爬虫不能保证所有博客都可以兼容，只是提供一个思想和思路，其他网站请自行分析。# xiaorui.cc爬虫说明
